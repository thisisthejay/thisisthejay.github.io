The trick with static sites is balancing structure with simplicity.

A practical setup:

1. Keep post metadata in frontmatter.
2. Generate index pages from file-system content.
3. Reuse one card style in multiple places (home + blog list).

This keeps publishing straightforward while still giving you room to expand features later.
