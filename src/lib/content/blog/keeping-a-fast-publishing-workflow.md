A good workflow should remove friction, not add it.

For me, that means every post should be publishable by changing one markdown file and pushing to Git.

### Useful guardrails

- Use a required frontmatter template.
- Keep styles centralized in your post page.
- Link related posts at the bottom to improve discovery.

This setup gives you consistency without sacrificing speed.
